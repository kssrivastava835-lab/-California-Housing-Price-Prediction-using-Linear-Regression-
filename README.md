# -California-Housing-Price-Prediction-using-Linear-Regression-
“An end-to-end machine learning project using a Scikit-learn pipeline with feature scaling and linear regression to predict California housing prices and evaluate performance using MSE.”

This project is about building a machine learning setup from start to finish, using that California Housing dataset to guess house prices. I went with Scikit-learn because it makes things easier, especially the Pipeline part which ties everything together. Like, you put in StandardScaler to normalize the features, and then Linear Regression to actually predict stuff. It feels straightforward, but I am not totally sure if its the best for everything.
